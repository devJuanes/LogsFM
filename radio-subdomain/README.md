# Radio Subdomain - radio.logsfm.com

App optimizada para móviles que muestra solo el reproductor de radio.

## Uso

Este sitio está diseñado para ser cargado en una WebView en la app Android.

## Variables de Entorno

```env
NEXT_PUBLIC_STREAM_URL=http://localhost:8000/stream
NEXT_PUBLIC_API_URL=http://localhost:8001
```

## Desarrollo

```bash
npm install
npm run dev
```

## Producción

```bash
npm run build
npm start
```
